## RandomCanteen

A WeChat mini-program for random selecting a canteen

![](https://raw.githubusercontent.com/SuXY15/MyPic/master/RandomCanteen/demo.jpg)

### Support places

+ 清华大学
+ 清华李兆基
+ 清华其他
+ 北京大学
